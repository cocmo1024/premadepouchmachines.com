The inner filter and hanger define usability

Dose position, seal cleanliness and hanger placement affect brewing. Acceptance should include opening and cup-use tests, not only pack appearance.

Outer envelopes carry the barrier load

Specify laminate, nitrogen, code and residual-oxygen criteria. Test the completed inner and outer pack as one product.
