What the project must solve

01

Independent feeders can each count correctly while a missing discharge event still creates an incomplete kit.

02

Similar parts can be loaded into the wrong feeder unless changeover and line-clearance controls are defined.

03

A total checkweight may not distinguish two different combinations with the same mass.

Buyer analysis

Define the recipe as a bill of materials

List every component, quantity, weight range, geometry and feeder assignment. State whether kits can contain alternatives and how the operator confirms the loaded part number.

Select verification by failure mode

Counting confirmation, feeder-discharge sensors, vision, checkweighing and barcode controls detect different errors. The verification plan should target the mistakes that matter for the product and customer.

Run deliberate challenge tests

During acceptance, create missing-part, double-feed, jam and wrong-recipe conditions. Verify detection, reject handling, alarm recovery and prevention of rejected kits returning to good output.
